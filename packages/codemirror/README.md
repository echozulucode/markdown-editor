# @echozedlabs/codemirror

CodeMirror 6 source + hybrid Markdown editing surface (with a frontmatter properties widget) for the [echozed markdown editor](https://github.com/echozulucode/markdown-editor).

Most consumers should install [`@echozedlabs/react`](https://www.npmjs.com/package/@echozedlabs/react), which wires this view into the full multi-mode editor. Use this package directly only if you're building a custom host.
