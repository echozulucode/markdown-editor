# @echozedlabs/core

Markdown parsing/serialization core for the [echozed markdown editor](https://github.com/echozulucode/markdown-editor): a byte-stable Markdown codec (parse → serialize is byte-identical), frontmatter split that preserves bytes verbatim, and the shared types used across the editor packages.

Most consumers don't use this directly — install [`@echozedlabs/react`](https://www.npmjs.com/package/@echozedlabs/react) for the full editor. Use `@echozedlabs/core` if you only need the headless codec.
